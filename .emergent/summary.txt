**PRODUCT REQUIREMENTS:**

**1. Public Pages:**
   - Home, About SESI, Executive Committee, Office Bearers (Year-wise), Members Directory (filterable), Membership (Eligibility, Types, Apply Online), Education (Courses, Fellowships, Workshops), Conferences & Events (Upcoming, Past), Publications, Gallery, News & Announcements, Contact Us.

**2. Admin Panel (Core Requirement):**
   - Secure login with role-based access (Admin/Editor).
   - Full CRUD (Create, Read, Update, Delete) for Committee members, Events, Members, News, and Gallery images.
   - Year-based tagging for committee members.
   - A workflow to review membership applications (view details, download documents, approve/reject).
   - SEO management fields for every page.

**3. Membership Registration Form ( equivalent):**
   - Detailed form with fields for personal details, addresses, and professional qualifications.
   - Required document uploads (MBBS, Orthopedic, State Registration certificates).
   - Dynamic dependent dropdowns for State -> District.
   - Both client-side and server-side validation.

**4. Post-Approval Workflow:**
   - On application approval, automatically generate a unique membership registration number.
   - Automatically generate a digital certificate (PDF).
   - Send an email to the applicant with the membership number and the certificate attached.
   - Create a member profile in the system, which should be viewable in the admin panel.

**5. Database:**
   - A normalized schema to store all data for users, members, committees, events, applications, etc.

**6. UI/UX:**
   - Clean, professional, academic look with a neutral color palette (white, light grey, deep blue/green accents).
   - Easy navigation, especially for senior surgeons.
